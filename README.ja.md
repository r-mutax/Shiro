# Shiro 言語仕様書

[English version (README.md)](./README.md)

Shiro は、64ビット符号付き整数を扱う非常にシンプルな手続き型言語およびそのコンパイラです。

## 1. 基本仕様
*   **データ型**: 暗黙的に **64ビット符号付き整数（64-bit signed integer）** のみを取り扱います。
*   **エントリーポイント**: ソースコード中の文は、アセンブリ出力時に暗黙的に `main` 関数内の処理としてコンパイルされます。
*   **プログラムの戻り値**: プログラム（関数）の最後に評価された文（Statement）の値が、プログラムの終了コード（Exit Code）になります。

---

## 2. 構文規則（EBNF表現）

```ebnf
Program            ::= Statement*
Statement          ::= ExpressionStatement | VariableDeclareStatement

ExpressionStatement        ::= Expression ";"
VariableDeclareStatement   ::= "let" Identifier ";"

Expression         ::= Assign
Assign             ::= Shift [ "=" Assign ]
Shift              ::= AddSub ( ( "<<" | ">>" ) AddSub )*
AddSub             ::= MulDivMod ( ( "+" | "-" ) MulDivMod )*
MulDivMod          ::= Primary ( ( "*" | "/" | "%" ) Primary )*
Primary            ::= Number | Identifier | "(" Expression ")"

Identifier         ::= [a-zA-Z_][a-zA-Z0-9_]*
Number             ::= [0-9]+
```

---

## 3. 演算子の優先順位と結合規則

下に行くほど優先順位が高くなります。代入演算子（`=`）は**右結合**、それ以外の二項演算子は**左結合**です。

| 優先順位 | 演算子 | 結合性 | 説明 | 例 |
| :--- | :--- | :--- | :--- | :--- |
| 1 (低) | `=` | 右結合 | 代入 | `y = x = 10` |
| 2 | `<<`, `>>` | 左結合 | ビット左シフト、ビット右シフト | `1 << 2` |
| 3 | `+`, `-` | 左結合 | 加算、減算 | `x + 5` |
| 4 | `*`, `/`, `%` | 左結合 | 乗算、除算、剰余算 | `10 % 3` |
| 5 (高) | `( )` | なし | グループ化（括弧） | `(2 + 3) * 4` |

## 4. 変数仕様
*   **宣言**: `let <変数名>;`
    *   変数宣言時、自動的に値 `0` に初期化（ゼロ初期化）されます。
*   **代入**: `<左辺値> = <式>`
    *   代入は「式（Expression）」として扱われ、代入された値自身を評価値として返します。右結合であるため、`y = x = 10` のような連続した代入が可能です。
    *   現時点で代入の左辺値（Lvalue）として使用できるのは宣言済みの変数のみです。
*   **スコープ**: 
    *   変数は宣言されたスコープ（現在は関数スコープのみ）の中で有効です。
*   **セマンティクスチェック（検証規則）**:
    *   **二重宣言の禁止**: 同じスコープ内で同名の変数を複数回宣言することはできません。
    *   **未定義変数の使用禁止**: 宣言されていない、あるいは宣言される前の行で変数を使用したり代入したりすることはできません。
    *   **左辺値の検証**: 代入の左辺は代入可能なオブジェクト（現在は変数のみ）である必要があります。

## 5. コード例

### 基本的な演算と変数
```rust
let x;         // x を 0 で初期化
let y;         // y を 0 で初期化
x = 5;         // x に 5 を代入
y = x + 2;     // x + 2 (7) を y に代入
y;             // 最終行の評価値 (7) が終了コードになる
```

### 連続代入（右結合の検証）
```rust
let x;
let y;
y = x = 10;    // x と y の両方に 10 が代入される
x + y;         // 最終評価値: 20
```

### 代入式自体の評価値
```rust
let x;
(x = 5) + 5;   // 代入式 (x = 5) 自体が 5 と評価され、それに 5 を加算する。最終評価値: 10
```
